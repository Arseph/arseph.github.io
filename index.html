aron@cotabato

I'm a developer who likes owning the entire stack — schema design, API architecture, frontend state, server configuration, and the deployment pipeline in between. Most of my work lives where reliability matters: government HR systems, payroll engines, and public service portals.

Lately that means Laravel 12 APIs with Sanctum auth, Vue 3 dashboards, and React Native apps talking to the same backend. I run my own Ubuntu dev server, tune Nginx by hand, and care about the details — from database indexes to the exact spacing on a payroll PDF.
